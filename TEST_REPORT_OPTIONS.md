## ✅ 功能檢查清單

### 1. 📦 附加選項庫（localStorage）
- [x] 預設選項已初始化
- [x] 6 個預設選項：
  - 🌶️ 辣度（不辣、小辣、中辣、大辣）
  - 🍚 飯量（正常、大碗 +$2.00）
  - 🧂 口味（蜜汁、椒鹽、原味）
  - 🥬 配菜（高麗菜、泡菜、海帶、豆干）
  - 🥤 附加飲料（不加、冬瓜茶 +$2.00、紅茶 +$2.00、奶茶 +$3.00、可樂 +$2.50）
  - 💧 水（不需要、瓶裝水 +$1.00）

### 2. 🛒 前台：點餐流程測試

#### A. 選項彈窗觸發
- [ ] 點擊有附加選項的菜單項目
- [ ] 彈出附加選項 Modal
- [ ] Modal 顯示商品圖片/Emoji
- [ ] Modal 顯示商品名稱和基本價格

#### B. 選項選擇
**必選項目（辣度、飯量、口味、配菜）：**
- [ ] 顯示 "（必選）" 標籤
- [ ] 預設選擇第一個選項
- [ ] 可以切換選擇（單選）
- [ ] 未選擇時無法加入購物車

**可選項目（附加飲料、水）：**
- [ ] 不顯示 "（必選）" 標籤
- [ ] 可以不選擇
- [ ] 可以切換選擇（單選）

#### C. 價格計算
- [ ] 基本價格顯示正確
- [ ] 選擇附加選項後，價格即時更新
- [ ] "目前價格" 顯示正確總價
- [ ] 選項總結區域顯示所有選擇
- [ ] 附加費用顯示 "+$X.XX" 格式

#### D. 加入購物車
- [ ] 點擊「取消」按鈕關閉 Modal
- [ ] 點擊「加入購物車」按鈕成功添加
- [ ] 購物車顯示商品名稱
- [ ] 購物車顯示所有選擇的選項
- [ ] 購物車顯示正確的總價

### 3. 🔧 後台：選項管理測試

#### A. 查看選項庫
- [ ] 訪問後台管理系統
- [ ] 登入（密碼：admin123）
- [ ] 點擊側邊欄「⚙️ 附加選項管理」
- [ ] 顯示所有現有選項

#### B. 新增選項
- [ ] 點擊「➕ 新增選項」按鈕
- [ ] 填寫選項名稱（如：甜度）
- [ ] 選擇類型（單選/多選）
- [ ] 設定是否必選
- [ ] 添加選項值（如：正常甜、半糖、微糖、無糖）
- [ ] 設定每個選項值的價格
- [ ] 儲存成功

#### C. 編輯選項
- [ ] 點擊現有選項的「編輯」按鈕
- [ ] 修改選項名稱
- [ ] 修改選項值
- [ ] 修改價格
- [ ] 儲存成功
- [ ] 前台即時更新

#### D. 刪除選項
- [ ] 點擊「刪除」按鈕
- [ ] 顯示確認對話框
- [ ] 確認刪除後成功移除
- [ ] 前台不再顯示該選項
